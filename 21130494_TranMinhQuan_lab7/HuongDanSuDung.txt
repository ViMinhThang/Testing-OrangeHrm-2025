Bởi vì tính phức tạp của trang web khiến cho việc dùng Selenium IDE gặp cực nhiều khó khăn nên 
em xin phép cô được viết code bằng Python
Phần script nằm trong thư mục src

Link google sheet chứa test case: https://docs.google.com/spreadsheets/d/1k8jzEYfhNT1-jEnd5EEYnAiihefAVyVbmXeLC71PLAU/edit?gid=1106634420#gid=1106634420

Yêu cầu:
+ Python >= 3.10
+ Trình duyệt Chrome

Các bước chạy (tất cả đều thực hiện trên Terminal):
* Bước 1 - tạo môi trường ảo:
	- python -m venv venv
* Bước 2 - kích hoạt môi trường ảo:
	- Window: venv\Scripts\activate
	- Mac/Linux: source venv/bin/activate
Khi kích hoạt thành công sẽ có (venv) xuất hiện tại đầu dòng lệnh
* Bước 3 - cài đặt thư viện cần thiết (gồm Selenium, Pytest và các thư viện liên quan)
	- pip install -r requirements.txt
* Bước 4 -  Chạy chương trình automation test
	- pytest -v .\src\21130494_TranMinhQuan_Lab7.py
* Bước 5 - thoát môi trường ảo:
	- deactivate
